Comment utiliser BAND2BINKS ?

  • Entrez une URL Bandcamp dans le champ prévu.
  • Cliquez sur "Télécharger" pour lancer le processus.
  • Une liste des fichiers MP3 apparaîtra sous forme de cartes en cliquant sur la flèche à gauche du Pack.
  • Cliquez sur un fichier individuel pour le télécharger ou utilisez "Télécharger tout" pour récupérer un dossier compressé.
  • Outil à utiliser pour un apprentissage du développement web uniquement. Ne pas utiliser chez soit illégalement dans le but de faire glow up vos Dj Set!
